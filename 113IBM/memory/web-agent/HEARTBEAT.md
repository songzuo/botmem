# HEARTBEAT.md - Web Agent (爬虫代理)

## 职责
- 网页爬取和信息收集
- 新闻监控
- 数据采集
- 内容分类存储

## 心跳频率
- 每 1 小时执行爬取任务

## 定时任务

### 每小时执行
1. 执行定时爬取任务
2. 处理爬取数据
3. 更新数据索引

### 每天执行 (07:00, 19:00)
1. 生成爬取报告
2. 数据去重和清洗
3. 更新知识库

## 爬取目标
- 行业新闻
- 技术博客
- 竞争对手网站
- 用户指定 URL
