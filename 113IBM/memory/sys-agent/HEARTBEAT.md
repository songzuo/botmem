# HEARTBEAT.md - Sys Agent (系统代理)

## 职责
- 监控系统资源 (CPU/内存/磁盘)
- 管理 Gateway 和关键服务状态
- 服务器健康检查
- 异常告警

## 心跳频率
- **每 15 分钟** 检查系统状态 (关键基础设施)

## 定时任务

### 每 15 分钟执行
1. 检查 CPU/内存/磁盘使用率
2. 检查 Gateway 端口 (18789) 状态
3. 检查 PostgreSQL (5432) 状态
4. 检查代理端口 (1080) 状态
5. 记录系统指标到日志

### 每小时执行
1. 检查 bot2-bot7 服务器连接状态
2. 扫描系统日志发现异常
3. 更新系统状态报告

### 每天执行 (06:00)
1. 生成系统健康日报
2. 检查磁盘空间趋势
3. 清理临时文件

## 监控阈值
- CPU 使用率 > 80%: 警告
- 内存使用率 > 85%: 警告
- 磁盘使用率 > 90%: 严重警告
- Gateway 离线：立即告警

## 状态追踪
- 上次检查：待初始化
- 下次报告：待初始化
- 完成度：0%

## 异常处理
- 发现异常立即通知主代理
- 记录详细日志到 memory/sys-agent/logs/
- 关键服务离线时尝试自动重启
