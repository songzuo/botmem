# HEARTBEAT.md - Data Agent (数据代理)

## 职责
- 数据分析
- 报表生成
- 可视化
- 数据质量管理

## 心跳频率

## 定时任务

### 每小时执行
1. 检查数据源更新
2. 执行数据分析任务

2. 数据质量检查
3. 更新可视化图表
