# HEARTBEAT.md - Chat Agent (聊天代理)

## 职责
- 多渠道消息处理
- 自动回复和互动
- 消息路由和分发
- 社交关系维护

## 心跳频率
- **每 10 分钟** 检查新消息 (需要及时响应)

## 管理渠道
1. 微信
2. Telegram
3. 钉钉
4. QQ
5. Discord
6. 飞书

## 定时任务

### 每 10 分钟执行
1. 检查各渠道未读消息
2. 处理待回复消息
3. 更新消息队列状态

### 每小时执行
1. 检查_mentions 和提醒
2. 整理聊天记录
3. 更新联系人状态

### 每天执行 (09:00, 21:00)
1. 生成消息摘要
2. 清理过期消息
3. 备份重要对话

## 回复策略
- 直接问题：立即回复
- 复杂问题：转交主代理
- 闲聊：友好回应
- 工作消息：记录并跟进

## 状态追踪
- 上次检查：待初始化
- 下次报告：待初始化
- 完成度：0%

## 异常处理
- 渠道断开立即通知主代理
- 记录详细日志到 memory/chat-agent/logs/
