# HEARTBEAT.md - Memory Agent (记忆代理)

## 职责
- 管理所有记忆文件的整理和归档
- 每日记忆摘要生成
- 清理过期记忆
- 维护 MEMORY.md 长期记忆

## 心跳频率
- 每 1 小时检查记忆更新
- 每 24 小时生成记忆摘要
- 每 7 天进行记忆归档

## 定时任务

### 每小时执行
1. 扫描 memory/ 目录下新增/修改的文件
2. 提取关键信息到 shared/ 共享区
3. 更新记忆索引

### 每天执行 (02:00)
1. 生成昨日记忆摘要
2. 更新 MEMORY.md 长期记忆
3. 清理临时文件

### 每周执行 (周日 03:00)
1. 归档旧记忆文件
2. 生成周度记忆报告
3. 优化记忆结构

## 状态追踪
- 上次检查：待初始化
- 下次报告：待初始化
- 完成度：0%

## 记忆管理规则
1. 保持 memory/ 目录下文件不超过 90 天
2. 重要决策永久保存到 MEMORY.md
3. 项目相关记忆关联到对应代理目录

## 异常处理
- 发现记忆冲突时通知主代理
- 记录详细日志到 memory/memory-agent/logs/
