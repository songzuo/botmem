## 测试策略

### Mock 数据生成器

为了支持全面的测试，创建了以下 Mock 数据生成器：

1. **`createMockWorkflow()`** - 创建基本的三节点工作流（Start → Agent → End）
2. **`createConditionWorkflow()`** - 创建包含条件分支的工作流
3. **`createParallelWorkflow()`** - 创建并行执行的工作流

### 测试技术

- **Vitest** 测试框架
- **describe/it** 测试套件结构
- **beforeEach** 测试隔离和清理
- **expect/assertions** 丰富的断言验证
- **async/await** 异步测试支持
