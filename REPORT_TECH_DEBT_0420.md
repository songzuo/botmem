## 2️⃣ TypeScript any 类型使用（447处）

### 分布情况

| 目录 | 数量 | 典型问题 |
|------|------|----------|
| src/lib/knowledge/ | ~40处 | 向量存储、文档处理 |
| src/lib/performance/ | ~35处 | 批处理、缓存 |
| src/lib/search/ | ~20处 | 搜索建议 |
| src/components/ | ~50处 | 组件 Props |
| src/app/ | ~30处 | 页面数据 |
| 测试文件 | ~80处 | Mock 对象 |

### 重点治理建议

1. **优先级高 - 向量存储 (knowledge/)**
   - `vector-store.ts`: 70行 `client: any = null`
   - `document-pipeline.ts`: 301-302行 JSON 响应类型
   - `rag-qa.ts`: 179行 API 响应类型

2. **优先级中 - 性能工具 (performance/)**
   - `batch-request.ts`: 多个 `any` 参数和返回值
   - `offline-storage.ts`: 数据存储类型

3. **优先级低 - 测试文件**
   - 可接受使用 `any` 进行 mock

### 建议措施

- 使用 `unknown` 替代 `any` 增加类型安全
- 定义明确的接口类型
- 使用 `zod` 或 `io-ts` 进行运行时验证
