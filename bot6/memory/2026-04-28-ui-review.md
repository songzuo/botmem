## 5. 改进建议

### 🔴 高优先级

1. **Select 组件键盘可访问性**
   - 添加键盘导航支持（上下箭头切换选项）
   - 添加 `role="listbox"`, `role="option"`
   - 添加 `aria-expanded`, `aria-selected`

2. **Tabs 组件 ARIA**
   - TabsList 添加 `role="tablist"`
   - TabsTrigger 添加 `role="tab"`, `aria-selected`
   - TabsContent 添加 `role="tabpanel"`, `aria-labelledby`

3. **Modal 组件 ARIA**
   - 添加 `role="dialog"`
   - 添加 `aria-modal="true"`
   - 添加 `aria-labelledby` 关联标题元素

### 🟡 中优先级

4. **Switch 组件**
   - 改用 `<button role="switch">` 实现
   - 添加 `aria-checked` 属性
   - 添加暗色模式支持

5. **Progress 组件**
   - 添加 ARIA 属性：`aria-valuenow`, `aria-valuemin`, `aria-valuemax`
   - 添加暗色模式前景色

6. **Badge 组件**
   - 为所有变体添加 `dark:` 样式

### 🟢 低优先级

7. **Label 组件**
   - 添加自定义 Props 类型定义
   - 添加暗色模式支持

8. **EmptyState**
   - 为图标容器添加明确的暗色样式
