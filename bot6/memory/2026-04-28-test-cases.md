## 测试用例列表

### 1. 用户注册 - 正常流程

**测试ID**: TC-AUTH-001  
**测试类型**: 正常路径  
**端点**: `POST /api/auth/register`  
**前置条件**: 无（未登录状态）  
**测试步骤**:
1. 提交有效的邮箱、密码、用户名
2. 验证响应状态码 200
3. 检查返回的 user 对象包含 id、email、name
4. 验证 token 字段存在且非空

**预期结果**:
- 注册成功，返回用户信息和 JWT token
- success 字段为 true
