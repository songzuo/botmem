#### 1. WebSocketMonitor 类实现 ✅

**文件**: `src/lib/monitoring/websocket-monitor.ts`

功能:
- 自动追踪 Socket.IO 连接延迟
- 监控断开重连次数
- Ping-pong 延迟测试
- 自动上报到性能监控系统
- 支持客户端和服务端监控

主要方法:
- `trackSocketClient(socket, namespace)` - 监控客户端连接
- `trackSocketServer(io, namespace)` - 监控服务端
- `getMetrics(namespace?)` - 获取指标
- `getStats()` - 获取统计信息
- `getLatencyHistory(namespace?)` - 获取延迟历史
- `stopTracking(namespace)` - 停止监控

### 配置说明

```typescript
const config = {
  pingInterval: 5000,           // Ping 间隔 (ms)
  latencyWarningThreshold: 200, // 延迟警告阈值 (ms)
  latencyCriticalThreshold: 500, // 延迟严重阈值 (ms)
  autoReport: true,             // 自动上报到性能监控
  verbose: false,               // 详细日志
  maxHistoryLength: 100,        // 历史记录最大长度
};
```

### 使用示例

```typescript
// 服务端
import { setupWebSocketMonitoring } from "@/lib/websocket/monitoring-middleware";

setupWebSocketMonitoring(io);

// 客户端
import { useWebSocketMonitoring } from "@/lib/websocket/monitoring-middleware";

const cleanup = useWebSocketMonitoring(socket, "notifications");

// 组件卸载时清理
cleanup();

// 获取统计
import { getWebSocketStats } from "@/lib/websocket/monitoring-middleware";

const stats = getWebSocketStats();
console.log(`Active connections: ${stats.activeConnections}`);
console.log(`Average latency: ${stats.avgLatency}ms`);
```
