### 6. 严重问题详情

#### 6.1 日语JSON文件损坏 ❌

**位置**: `src/i18n/messages/ja.json` 第176行

**问题**:
```json
{ "question": "<minimax:tool_call>
  "answer": "はい、初回無料のご相談を行っています..."
}
```

**原因**: 包含未闭合的字符串和控制字符 `<minimax:tool_call>`，这看起来像是AI生成过程中的残留标记。

**影响**: JSON无法解析，日语国际化完全失效。

**修复建议**:
1. 从备份恢复（存在 `ja.json.bak`）
2. 或从英语版本重新翻译
3. 确保所有字符串正确闭合

#### 6.2 西班牙语混入中文 ❌

**位置**: `src/i18n/messages/es.json` 第29行

**问题**:
```json
"cta1": "了解更多"  // 这是中文，应该是 "Saber Más"
```

**影响**: 西班牙语用户在首页看到中文按钮。

**修复建议**: 立即修正为 `"cta1": "Saber Más"`

#### 6.3 韩语混入中文/日文 ❌

**位置1**: `src/i18n/messages/ko.json` 第56行
```json
"description": "설계에서 구현까지,高性能な現代 웹사이트와 웹 애플리케이션 구축"
// "高性能な"是日文，应该是 "고성능의"
```

**位置2**: `src/i18n/messages/ko.json` 第98行
```json
"description": "피드백에 기반한 신속한 반복과 최적화로 제품과 경험不断完善"
// "不断完善"是中文，应该是 "지속적으로 개선"
```

**影响**: 韩语用户看到混合语言，专业度受损。
