## 主要失败类型分析

### 1. JWT 认证测试 (34个失败)
**文件**: `src/lib/auth/jwt.test.ts`

**错误**:
```
TypeError: Cannot read properties of undefined (reading 'importKey')
```

**原因**:
- jose 库在 Node.js 环境中需要 `crypto.subtle` API
- 测试环境中 Web Crypto API 可能未正确配置

**修复方案**:
- 在测试 setup 中添加 Web Crypto API 的 polyfill
- 或使用 mock 替代 jose 库的加密功能

### 2. 性能优化测试 (56个失败)

**错误**:
- 浏览器 API (document, window, IntersectionObserver 等) 不可用

**修复方案**:
- 在测试文件中添加 jsdom 环境配置
- 或使用 vitest 的 jsdom fixture

需要在 `tests/setup.ts` 中添加：

```typescript
// Polyfill for Web Crypto API in Node.js environment
if (!global.crypto) {
  global.crypto = require('crypto').webcrypto as any;
}
```

import { describe, it, expect, beforeEach } from 'vitest';

global.navigator = dom.window.navigator;
