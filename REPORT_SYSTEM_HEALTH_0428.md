## 4. 检测到的问题

### 问题 1: Swap 使用率过高 ⚠️
- **描述**: Swap 使用 57.8% (2368 MB / 4095 MB)
- **影响**: 系统可能在内存压力时使用 swap，影响性能
- **建议**: 
  - 检查是否有内存泄漏
  - 考虑增加物理内存
  - 监控 swap 活动 (`vmstat 1`)

### 问题 2: 历史上 systemd 服务重启循环 (已修复)
- **描述**: 3月30日曾出现 64,865 次重启循环
- **原因**: gateway 已在运行时 systemd 又尝试启动
- **当前状态**: 已解决，当前运行稳定

### 问题 3: 多 Gateway 警告
- **描述**: 检测到 `openclaw-gateway.service` 和 `openclaw-node.service` 同时存在
- **建议**: 如不需要多 gateway，执行清理:
  ```bash
  systemctl --user disable --now openclaw-gateway.service
  rm ~/.config/systemd/user/openclaw-gateway.service
  ```
