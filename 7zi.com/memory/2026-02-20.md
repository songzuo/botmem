## 系统特性

### 自主任务调度
- 支持定时执行任务
- 任务包括：系统报告生成、Telegram报告发送、服务器协调、Twitter内容检查、使用CodeBuddy工作
- 可扩展的任务定义和调度配置

### CodeBuddy 免费token管理
- 5小时工作周期，确保充分利用免费token
- 休息阶段管理，确保token重置
- 每次任务执行时间限制在30分钟内，避免超时

### 系统资源监控
- 定期监控系统资源使用情况
- 记录和分析系统运行时间、负载和内存使用
- 提供状态检查和错误报告功能

### 错误处理和可靠性
- PID文件管理，防止重复启动
- 完整的错误处理机制
- 详细的日志记录和状态报告

### 服务器协调管理
- 管理所有6台服务器的状态和任务执行
- 自动检查服务器在线状态
- 自动启动/停止OpenClaw服务
- 根据服务器角色分配和执行任务
- 支持多种任务类型：algora-bounty, polymarket-bounty, data-analysis, global-search, csv-profiling, task-execution等

## 使用说明

### 启动 OpenClaw 自主工作流程系统
```bash
cd /root/.openclaw/workspace
./start-workflow.sh
```

### 启动 OpenClaw 服务器协调系统
```bash
cd /root/.openclaw/workspace
./start-openclaw-coordinator.sh
```

### 停止 OpenClaw 自主工作流程系统
```bash
cd /root/.openclaw/workspace
./stop-workflow.sh
```

### 停止 OpenClaw 服务器协调系统
```bash
cd /root/.openclaw/workspace
./stop-openclaw-coordinator.sh
```

### 检查系统状态
```bash
cd /root/.openclaw/workspace
./check-workflow.sh
```
