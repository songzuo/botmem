## 子代理协作模式

### 任务分配流程
1. Observer 发现问题/机会
2. 报告给 Commander
3. Commander 派遣合适的子代理
4. 子代理执行任务
5. 结果记录到 memory

### 优先级
- P0: 服务完全不可用 - 立即处理
- P1: 核心功能受损 - 15分钟内
- P2: 非核心功能问题 - 1小时内
- P3: 轻微问题 - 4小时内
