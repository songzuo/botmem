## 🔍 发现的行为说明

### Rooms.ts

1. **maxParticipants 验证:** 当前实现不验证负数或零值，测试文档了此行为
2. **重复房间 ID:** 返回现有房间而非创建新房间
3. **项目房间:** 默认不自动清理 (autoCleanupMinutes = 0)
4. **封禁权限:** 可以封禁包括所有者在内的任何角色用户

### Permissions.ts

1. **权限过期:** 支持 expiresAt，过期后自动拒绝
2. **角色层次:** owner > admin > moderator > member > guest
3. **全局角色:** 独立于房间特定角色
4. **封禁影响:** 封禁用户自动撤销所有权限

### Message-store.ts

1. **消息覆盖:** 相同 ID 的消息会被覆盖
2. **离线消息队列:** 超过 maxOfflineMessages 时驱逐最旧消息
3. **TTL:** 离线消息过期后不再返回
4. **软删除:** 默认历史查询中排除已删除消息
