### 5.1 反馈处理闭环流程

```
┌─────────────────────────────────────────────────────────────┐
│                    用户反馈处理闭环                          │

│                                                             │

│      ▼                                                      │

│      │               │                │                │     │
│      │               ▼                ▼                ▼     │
│      │          [管理员通知]    [管理员通知]    [满意度调查]  │
│      │                                                    │
│      ▼                                                    │
│  [用户查询进度] ◀─────────────────────────────────────────┘
│      │
│      ▼
│  [满意度评价] → [评价奖励/积分]
│
└─────────────────────────────────────────────────────────────┘
```

### 5.2 自动化触发规则

| 触发条件 | 执行动作 | 优先级 |
