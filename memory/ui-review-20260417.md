## 5. 代码质量发现

### 5.1 Bug: MobileLayout.tsx
```tsx
// 第 61-64 行存在问题
const toggleSidebar = () => setIsOpen(!isOpen)  // ❌ setIsOpen 未定义

// 应该使用:
const toggleSidebar = () => setIsSidebarOpen(prev => !prev)
```

1. **globals.css 冗余**: 存在大量硬编码颜色值，未完全使用 CSS 变量

## 7. 设计系统建议

### 7.1 设计令牌 (Design Tokens)
建议建立完整的设计令牌系统：
```css
:root {
  /* 主色调 */
  --color-primary-50: #EEF2FF;
  --color-primary-500: #667EEA;
  --color-primary-600: #5A67D8;
  --color-primary-700: #4C51BF;
  
  /* 语义色 */
  --color-success: #48BB78;
  --color-warning: #ECC94B;
  --color-danger: #F56565;
  
  /* 间距 */
  --spacing-1: 0.25rem;
  --spacing-2: 0.5rem;
  /* ... */
  
  /* 圆角 */
  --radius-sm: 0.25rem;
  --radius-md: 0.5rem;
  --radius-lg: 1rem;
  /* ... */
}
```

### 7.2 组件变体统一样式
当前 Button 组件变体：`primary | secondary | outline | ghost | danger | success`
建议统一到设计系统，确保色彩、间距、圆角一致。
