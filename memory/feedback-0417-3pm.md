- **现象**: 用户报告页面缓存刷新失效
- **影响**: 内容更新不及时，用户看到旧数据
- **发现时间**: 2026-04-11
- **状态**: 待修复

- **文件**: 多个组件中使用
- **现象**: React 19 兼容性警告
- **影响**: 未来可能功能异常
- **状态**: 待迁移

- **现象**: 主题切换时页面闪烁
- **用户反馈**: 影响使用体验
- **状态**: v1.14.0 已优化
