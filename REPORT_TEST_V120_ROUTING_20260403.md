## 五、已发现并修复的问题

在测试过程中发现并修正了以下预期行为差异：

1. **预算检查行为**: 当 dailyBudgetLimit 为 0 时，剩余预算返回 Infinity 而非 0
2. **路由器默认行为**: 即使所有模型被禁用，路由器仍尝试使用默认模型
3. **队列处理机制**: 队列项目被同步处理，立即执行
