# 2026-03-04 21:01 MiniMax Coding Plan 配置研究

## 用户提供的配置要点

### MiniMax Coding Plan 配置格式

**provider 配置:**
- name: minimax (自定义)
- type: anthropic-compatible
- base_url: https://api.minimaxi.com/anthropic
- api_key: 你的 Coding Plan API Key
- models:
  - id: MiniMax-M2.5
  - id: MiniMax-M2.1
  - id: MiniMax-M2

### 关键点
1. **Coding Plan API Key** 与普通 MiniMax API Key 不能互换
2. **base_url**: 必须用 `https://api.minimaxi.com/anthropic` (不是 api.minimax.chat)
3. **type**: anthropic-compatible

### 测试命令
```bash
curl https://api.minimaxi.com/anthropic/v1/messages \
  -H "Authorization: Bearer $MINIMAX_CODING_PLAN_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "MiniMax-M2.5",
    "messages": [
      {"role": "system", "content": "You are a helpful assistant."},
      {"role": "user", "content": "Write a Python function to sort an array."}
    ]
  }'
```

## 解决方案
将沙箱的完整配置（models + agents）复制到 bot6 和 bot7：

1. 获取沙箱配置:
   ```bash
   openclaw config get models > /tmp/sandbox_models.json
   openclaw config get agents > /tmp/sandbox_agents.json
   ```

2. 复制到目标机器:
   ```bash
   scp /tmp/sandbox_*.json root@<IP>:/tmp/
   ```

3. 应用配置:
   ```python
   current['models'] = sandbox_models
   current['agents'] = sandbox_agents
   ```

4. 重启 gateway
