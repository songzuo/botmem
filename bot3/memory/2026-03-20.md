## Optimizations Implemented

### 1. Enhanced `optimizePackageImports` Configuration

**Changes**:
- Added `lucide-react` to experimental.optimizePackageImports
- This enables tree-shaking for icon imports

**Configuration**:
```typescript
experimental: {
  optimizePackageImports: [
    'next-intl',
    '@sentry/nextjs',
    'zustand',
    'web-vitals',
    'lucide-react',  // NEW: Icon library optimization
  ],
}
```

**Expected Impact**:
- Reduced bundle size for lucide-react icons
- Faster initial load time
- Better tree-shaking

### 2. Compiler Console Removal

**Changes**:
- Added `compiler.removeConsole` for production
- Keeps error/warning logs for debugging

**Configuration**:
```typescript
compiler: {
  removeConsole: process.env.NODE_ENV === 'production' ? {
    exclude: ['error', 'warn'],
  } : false,
}
```

**Expected Impact**:
- Smaller bundle size (no console statements in production)
- Reduced runtime overhead
