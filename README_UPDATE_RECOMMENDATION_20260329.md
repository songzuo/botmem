````markdown
# React Compiler Configuration
# 可选功能：启用 React Compiler 以减少不必要的重新渲染 (20-40% 减少)
# 值: true | false (默认: false)

# 值: opt-in (只编译指定目录) | opt-out (编译除黑名单外的所有文件) | all (编译所有)
# 默认: opt-out
REACT_COMPILER_MODE=opt-out
```
````
