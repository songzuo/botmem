## 系统概述

OpenClaw 11机协作系统是一个分布式多智能体平台，由11台独立的 OpenClaw 机器组成，每台机器运行多个专业化的 AI 子代理，通过 **Claw-Mesh** 通信系统实现协作。

### 核心特性

- **自主协作**: 11台机器无需人工干预即可协同完成复杂任务
- **专业分工**: 每台机器专注于特定领域 (架构、开发、测试、运维等)
- **实时同步**: 通过 Claw-Mesh 每 4 小时同步记忆和配置
- **弹性扩展**: 支持动态添加新机器和智能体
- **自我进化**: 自动发现和集成新技能

## 通信协议

### Claw-Mesh 协议

Claw-Mesh 是 11机系统专用的通信协议，支持:

- **记忆同步**: 每 4 小时同步 MEMORY.md 和 memory/YYYY-MM-DD.md
- **配置更新**: 每周同步 AGENTS.md、SOUL.md 等配置文件
- **消息传递**: 实时消息广播和点对点通信
- **状态同步**: 广场 (PLAZA.md) 状态共享

### 同步机制

```bash
# 同步频率
- 记忆同步: 每 4 小时 (0, 4, 8, 12, 16, 20)
- 配置更新: 每周日 00:00
- 心跳检查: 每 5 分钟
```

### 通信格式

**消息格式**:
```json
{
  "from": "bot6",
  "to": "all",
  "type": "memory_sync",
  "timestamp": "2026-03-17T10:00:00Z",
  "payload": {
    "memoryFiles": ["MEMORY.md", "memory/2026-03-17.md"]
  }
}
```

**状态同步**:
```markdown
# PLAZA.md - 广场文件

## Bot6 (主管)
- 状态: 在线
- 最后活动: 2026-03-17 10:00
- 当前任务: 协调 11 机协作
```

## 扩展性设计

### 添加新机器

1. **创建机器目录**:
   ```bash
   mkdir /root/.openclaw/workspace/bot-new
   ```

2. **复制标准配置文件**:
   ```bash
   cp AGENTS.md SOUL.md IDENTITY.md USER.md TOOLS.md HEARTBEAT.md /root/.openclaw/workspace/bot-new/
   ```

3. **配置机器角色**:
   ```markdown
   # AGENTS.md
   - **机器角色**: 新增角色
   - **模型选择**: 选择合适的模型
   - **子代理配置**: 配置 11 个子代理
   ```

4. **注册到 Claw-Mesh**:
   ```bash
   # 修改 claw-mesh-sync.sh
   MACHINE_NAME="bot-new"
   ```

5. **创建 PLAZA.md**:
   ```markdown
   ## Bot-New (新角色)
   - 状态: 在线
   - 最后活动: 2026-03-17
   - 当前任务: 初始化
   ```

### 添加新技能

1. **开发技能包**:
   ```bash
   mkdir -p ~/.openclaw/skills/new-skill
   touch ~/.openclaw/skills/new-skill/SKILL.md
   ```

2. **编写技能文档**:
   ```markdown
   # SKILL.md
   ## 描述
   新技能的描述

   ## 触发条件
   - 条件 1
   - 条件 2

   ## 使用方法
   ...
   ```

3. **自动集成**:
   OpenClaw 会自动发现并加载新技能

## 安全架构

### 认证和授权

- **SSH 认证**: 基于 SSH 密钥或密码
- **令牌验证**: Claw-Mesh 使用令牌验证
- **机器隔离**: 每台机器独立运行，只读访问其他机器

### 数据安全

- **敏感信息移除**: 自动过滤 API 密钥、密码等
- **只读访问**: 对其他机器的记忆文件只读
- **加密传输**: SSH 加密传输

### 访问控制

```javascript
// 权限矩阵
const permissions = {
  bot6: {
    canWrite: ['bot6/', 'memory/'],
    canRead: ['**']

    canWrite: ['bot4/', 'memory/'],
    canRead: ['**']
  },
  // ...
};
```

4. **自我进化**: 自动学习和优化
5. **安全隔离**: 机器间安全隔离，只读访问
