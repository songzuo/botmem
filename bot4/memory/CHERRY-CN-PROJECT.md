# 🍒 Cherry Studio 中国版开发计划

> **项目目标**: 开发适合中国用户、中国文化、中国习惯的 Cherry Studio 版本
> **开发服务器**: bot4 (当前机器)
> **发布服务器**: bot7 (7zi.com)
> **协作机器**: 11 台 OpenClaw 机器 + 多个子代理

---

## 📊 项目概述

### 源项目信息
- **名称**: Cherry Studio
- **仓库**: https://github.com/CherryHQ/cherry-studio
- **技术栈**: Electron + React + TypeScript
- **许可证**: AGPL-3.0

### 核心特性
- ✅ 多 LLM 提供商支持
- ✅ 300+ AI 助手
- ✅ 多模型同时对话
- ✅ 文档处理（PDF、Office 等）
- ✅ WebDAV 文件管理
- ✅ MCP 服务器支持
- ✅ 跨平台（Windows/Mac/Linux）
