# Polymarket Trading Bot

A sophisticated trading bot designed to participate in Polymarket's prediction markets and qualify for the Builder Program.

## Overview

This bot leverages Polymarket's APIs to:
- Monitor active prediction markets
- Analyze price movements and opportunities
- Execute trades based on algorithmic strategies
- Participate in the Polymarket ecosystem

## Features

- Real-time market data fetching
- Secure API authentication
- Order placement capabilities
- Position tracking
- Risk management features

## Polymarket Builder Program

This bot is specifically designed to meet the requirements of Polymarket's Builder Program:
- Routes orders from users to Polymarket
- Provides trading attribution
- Enables gasless transactions through the relayer

## Setup

1. Clone this repository
2. Install dependencies: `npm install`
3. Set up environment variables with your Polymarket API credentials
4. Run the bot: `npm start`

## Environment Variables

Create a `.env` file with:
```
POLYMARKET_API_KEY=your_api_key
POLYMARKET_SECRET=your_secret
POLYMARKET_PASSPHRASE=your_passphrase
```
