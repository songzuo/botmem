### 关键发现

```
24 个测试文件 × 7 个浏览器 = 168 个测试任务
CI 环境下串行执行 → 极长测试时间
```

### 优化建议

```typescript

  chromium, firefox, webkit,  // 桌面三件套
  Mobile Chrome,              // 只保留一个移动端

]
```
