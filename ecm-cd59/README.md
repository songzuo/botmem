## 🌟 What is this? | 这是什么？

This repository contains the complete configuration and memory of **11 interconnected OpenClaw machines**, forming a powerful collective intelligence system. Each machine runs specialized agents with different capabilities, working together to solve complex problems, share knowledge, and continuously evolve.

本仓库包含了**11台互联OpenClaw机器**的完整配置和记忆，构成了一个强大的集体智能系统。每台机器运行着具有不同能力的专门化智能体，协同工作以解决复杂问题、共享知识并持续进化。

## 🚀 One-Click Deployment | 一键部署

Simply point your OpenClaw to this repository, and it will automatically:
- Download all 11 machine configurations
- Generate 11 specialized sub-agents
- Set up collaborative workflows
- Enable automatic memory synchronization
- Start self-evolution capabilities

只需将你的OpenClaw指向此仓库，它将自动：
- 下载全部11台机器的配置
- 生成11个专门化的子智能体
- 设置协作工作流
- 启用自动记忆同步
- 开启自我进化能力

```bash
# Clone this repository and let your OpenClaw auto-configure
git clone https://github.com/songzuo/botmem.git
cd botmem
# Your OpenClaw will detect the configuration and auto-evolve!
```
