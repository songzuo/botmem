# 2026-03-03 修复 QQ 机器人连接失败问题

## 问题描述
QQ 机器人持续连接失败，错误信息显示 `access_token missing from QQ response`，表明无法从 QQ 开放平台 API 获取访问令牌。

## 分析与解决方案
1. 检查配置文件发现 `clientSecret` 可能无效
2. 尝试更新为备用 `clientSecret` 后问题仍存在
3. 确认当前使用的 `appId` 和 `clientSecret` 组合无效
4. 决定暂时禁用 QQ 机器人以避免错误

## 修复操作
- 修改 `openclaw.json` 中 QQ 机器人配置：
  - 将 `enabled` 从 `true` 改为 `false`
- 重启 OpenClaw 服务

## 验证结果
- OpenClaw 服务正常运行
- QQ 机器人不再尝试连接，消除了错误
- Telegram 和 DingTalk 机器人功能正常

## 下一步建议
如需重新启用 QQ 机器人，需：
1. 申请有效的 QQ 开放平台凭证
2. 更新 `appId` 和 `clientSecret`
3. 重新启用 QQ 机器人配置

## 文件变更
- `/root/.openclaw/openclaw.json`：禁用 QQ 机器人
