## 版本号规则

OpenClaw 版本号格式: `YYYY.M.D` 或 `YYYY.M.D-N`

- `YYYY.M.D` - 正式版本 (如 `2026.3.2`)
- `YYYY.M.D-beta.N` - Beta 测试版 (如 `2026.3.2-beta.1`)

## 重要版本变化（待补充）

### v2026.x 系列主要特性

- **多通道支持**: WhatsApp, Telegram, Discord, Slack, Signal, iMessage, Google Chat 等
- **多代理路由**: 支持多个独立工作区和会话的代理
- **自动化**: Cron 任务、Webhooks、Heartbeat
- **沙箱隔离**: Docker 容器隔离执行
- **配置热重载**: 修改配置无需重启

### 安装最新版本

```bash
npm install -g openclaw@latest
```

### 查看当前版本

```bash
openclaw --version
# 或
openclaw status
```
