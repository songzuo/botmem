## OpenClaw Knowledge Base Maintenance

Periodically (every few days or before major config tasks), check if OpenClaw knowledge base needs updating:

1. Check latest npm version: `npm show openclaw version`
2. Compare with knowledge/openclaw/versions.md
3. If new version available, update versions.md

This ensures the knowledge base stays current with official releases.
