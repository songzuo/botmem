### 1. 缺失的翻译文件 ❌（优先级：高）

**问题**:

- 配置支持 6 种语言，但仅实现了 2 种（中文、英文）

- SEO 的 hreflang 标签指向不存在的语言版本

**工作量预估**: 8-12 小时（每种语言 2-3 小时）

---

### 2. 硬编码文本问题 ❌（优先级：高）

#### `src/components/Footer.tsx` - 严重硬编码

**问题组件**:

```tsx
// 快速链接（硬编码中文）
const quickLinks = [
  { name: "首页", href: "/" },
  { name: "关于我们", href: "/about" },
  { name: "团队成员", href: "/team" },
  { name: "博客", href: "/blog" },
  { name: "联系我们", href: "/contact" },
  { name: "Dashboard", href: "/dashboard" },
];

// 服务项目（硬编码中文）
const services = [
  { name: "网站开发", href: "#services" },
  { name: "品牌设计", href: "#services" },
  { name: "SEO 优化", href: "#services" },
  { name: "营销推广", href: "#services" },
  { name: "UI/UX 设计", href: "#services" },
  { name: "AI 解决方案", href: "#services" },
];

// 联系方式（硬编码中文）
const contactInfo = [
  { icon: "📧", label: "邮箱", value: "business@7zi.studio", href: "mailto:business@7zi.studio" },
  { icon: "🌐", label: "网站", value: "7zi.studio", href: "https://7zi.studio" },
  { icon: "📍", label: "地址", value: "中国", href: "#" },
];

// 底部栏（硬编码中文）
<Link href="/privacy">隐私政策</Link>
<Link href="/terms">服务条款</Link>
<Link href="/cookies">Cookie 政策</Link>

// 品牌描述（硬编码中文）
<p>
  由 11 位专业 AI 代理组成的创新数字工作室，提供网站开发、品牌设计、
  营销推广等全方位数字化服务。
</p>
```

**影响**:

- Footer 组件在所有页面上都显示中文，无法切换语言
- 破坏了国际化的一致性
- 用户看到混合语言界面（导航栏可切换，但 Footer 固定中文）

**解决方案**:

1. 在 `src/i18n/messages/zh.json` 和 `en.json` 中添加 footer 命名空间
2. 修改 `Footer.tsx` 使用 `useTranslations` hook
3. 更新所有硬编码文本为翻译键

**工作量预估**: 2-3 小时
